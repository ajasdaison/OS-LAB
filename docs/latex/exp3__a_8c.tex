\doxysubsection{Detailed Description}
Demonstrates process creation using fork(), exec(), and \doxylink{exp5_8c_a5707b25e46a5abb6a19f65efb29d0afb}{wait()} system calls. 

This program creates a child process using {\ttfamily fork()}. The child process then replaces itself with the {\ttfamily /bin/date} command using {\ttfamily execlp()}. The parent process waits for the child process to complete using {\ttfamily \doxylink{exp5_8c_a5707b25e46a5abb6a19f65efb29d0afb}{wait()}}. This demonstrates process control in Unix-\/based operating systems.

Creates a child process that executes the {\ttfamily date} command. 

The function performs the following\+:
\begin{DoxyItemize}
\item Calls {\ttfamily fork()} to create a new child process.
\item If {\ttfamily fork()} fails, it prints an error message and exits.
\item If {\ttfamily fork()} succeeds, the child process executes the {\ttfamily date} command using {\ttfamily execlp()}.
\item The parent process prints its PID and the child\textquotesingle{}s PID, then waits for the child to finish.
\end{DoxyItemize}
