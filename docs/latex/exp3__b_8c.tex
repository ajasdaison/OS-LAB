\doxysubsection{Detailed Description}
Demonstrates the use of {\ttfamily stat()} system call to retrieve file information. 

This program uses the {\ttfamily stat()} system call to obtain information about a file named {\ttfamily hello.\+txt}. If the file exists, it displays details such as\+:
\begin{DoxyItemize}
\item File mode (permissions) in octal format.
\item User ID of the file owner.
\item File size in bytes.
\end{DoxyItemize}

If the file does not exist, an error message is printed using {\ttfamily perror()}.

Retrieves and prints file information using {\ttfamily stat()}. 

The function attempts to retrieve file metadata for {\ttfamily hello.\+txt} using the {\ttfamily stat()} system call. If successful, it prints\+:
\begin{DoxyItemize}
\item {\ttfamily st\+\_\+mode} (file mode/permissions) in octal.
\item {\ttfamily st\+\_\+uid} (user ID of the file owner).
\item {\ttfamily st\+\_\+size} (file size in bytes).
\end{DoxyItemize}

If {\ttfamily stat()} fails (e.\+g., the file does not exist), an error message is displayed.
