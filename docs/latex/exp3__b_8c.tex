\doxysubsection{Detailed Description}
Program to retrieve and display file information using {\ttfamily stat()}. 

This program checks for the existence of "{}hello.\+txt"{} and displays its metadata, including file mode, owner user ID, and size. If the file does not exist, it prints an error message and skips further execution.
